BOLIS GOURMET - LANDING PAGE
=============================

📌 Descripción:
Esta es una landing page sencilla y atractiva creada para promocionar y vender bolis gourmet.
Incluye animaciones, efectos visuales y un botón de contacto directo por WhatsApp.

👨‍💻 Desarrollador: Marco Antonio Hernández Flores

🧩 Contenido del proyecto:
---------------------------
- index.html          ← Página principal del sitio.
- bolis.jpg           ← Imagen de bolis para galería.
- bolis2.jpg          ← Imagen adicional para galería.
- bolis3.jpg          ← Imagen adicional para galería.
- bolis-gourmet.jpg   ← Imagen principal destacada.
- README.txt          ← Este archivo con instrucciones y explicación.

🎨 Características del sitio:
-----------------------------
- Fondo animado con gradiente de colores.
- Efecto de nieve animada en todo el sitio.
- Diseño responsivo (compatible con móviles y PCs).
- Galería de imágenes con efecto zoom al pasar el cursor.
- Botón de WhatsApp con mensaje personalizado para pedidos.
- Sección destacada de sabores.

🚀 Instrucciones para subir a Netlify:
--------------------------------------
1. Entra a: https://app.netlify.com/drop
2. Arrastra esta carpeta completa o un archivo .ZIP con los archivos listados arriba.
3. Espera a que cargue y Netlify te dará un enlace público del sitio.
4. (Opcional) Cambia el nombre del sitio en la configuración para algo más personalizado.

📌 Notas:
---------
- Asegúrate de que el archivo principal se llame exactamente: index.html
- Las imágenes deben estar en el mismo nivel que index.html (no en subcarpetas).
- Puedes editar el HTML con cualquier editor como VS Code, Sublime Text o Notepad++.
